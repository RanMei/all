  • Canvas

    canvas是HTML5（HTML的第五个版本）中加入的一个新标签。前端开发者可以在canvas内绘制位图。

  • Canvas Animation

    面向对象编程：
    对象{
     属性：坐标，速度，尺寸，颜色......
     方法：前进，后退......
    }

    原理：根据数据绘制一帧 → 计算出下一帧的数据 → 绘制下一帧 → 周而复始
